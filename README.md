# Eduardo Santo — Portfolio

## About

A clean, minimal, single-page portfolio showcasing my software engineering career, notable projects, and technical skills — with a focus on backend systems, cloud architecture, and agentic AI.

## Tech Stack

- Static HTML5 / CSS3 / vanilla JavaScript
- Inter + JetBrains Mono typography (Google Fonts)
- Fully responsive design
- Scroll-reveal animations via IntersectionObserver
- Deployable on GitHub Pages

## Sections

- **Hero** — Introduction with terminal-style visual
- **Stats** — Key career highlights at a glance
- **Experience** — Timeline covering Amazon/Ring, Pretty Technical, and Globant
- **Notable Work** — Featured projects including AI agents, infrastructure migrations, and platform engineering
- **Skills & Tech Stack** — Programming languages, cloud, data, DevOps, and agentic AI
- **Contact** — Email, LinkedIn, GitHub, and downloadable CV
